What Is Prompt It?

Prompt It turns any AI prompt into a reusable tool called a Promptlet. If you keep using the same prompt structure and only swap the inputs, a Promptlet will save you time and keep your results consistent.

Stop copying and pasting. Create it once, use it everywhere.

The Four Core Patterns

Pattern 1: The Reviewer

You have: Items that need evaluation against consistent criteria.

The Promptlet: Embeds your rubric or checklist and applies it to each item.

  • Teacher grading essays against a rubric
  • HR screening resumes for required qualifications
  • QA checking bug reports for completeness
  • Editor reviewing articles for style guide compliance
  • Lawyer checking contracts for standard clauses
  • Developer reviewing pull requests for coding standards

Pattern 2: The Transformer

You have: Content in one format that needs to become another format.

The Promptlet: Defines the transformation rules once and applies them repeatedly.

  • Meeting notes ‚Üí structured action items with owners and deadlines
  • Customer calls ‚Üí CRM summaries
  • Technical documentation ‚Üí user-friendly guides
  • Raw data ‚Üí executive summaries
  • Interview transcripts ‚Üí key insights
  • Social media posts ‚Üí different platform formats
  • Long emails ‚Üí bullet-point summaries

Pattern 3: The Analyzer

You have: Raw information that needs structured analysis.

The Promptlet: Contains your analysis framework and applies it consistently.

  • Error logs ‚Üí root cause analysis
  • Survey responses ‚Üí thematic groupings
  • Medical symptoms ‚Üí differential diagnosis considerations

Power User Tips

Start Simple

Your first Promptlet should solve one clear, repetitive task. Avoid the Swiss Army knife on day one.

Iterate on the Prompt

Test on a few examples before running it on hundreds. Small refinements compound.

Be Specific

‚ÄúGrade this essay‚Äù ‚Üí inconsistent results. ‚ÄúGrade this essay using the attached rubric. Provide scores for thesis (0‚Äì25), evidence (0‚Äì25), organization (0‚Äì25), and mechanics (0‚Äì25), with specific examples for each.‚Äù ‚Üí reliable results.

Chain Promptlets

  1. Promptlet A: Extract action items from meeting transcript.
  2. Promptlet B: Generate calendar events from action items.

Version Your Prompts

Keep track of what works. Save prior versions when you improve a prompt in case you need to revert.

Use Context Files

If your prompt needs rubrics, style guides, or examples, attach them to the Promptlet rather than pasting them each time.

Batch Similar Items

Process similar items together. You‚Äôll spot patterns and can refine the prompt for the whole batch.

Keep Humans in the Loop

Promptlets handle the mechanical first pass. You keep the judgment calls.

Expectations

What Prompt It Is NOT

  • Not a chat thread. Prompt It runs each promptlet as a one-shot transformer‚Äîfollow-up runs don‚Äôt remember prior inputs or conversation history.
  • Not a replacement for expertise. It handles mechanical processing so your expertise can focus where it matters.
  • Not infallible. AI makes mistakes. Always review outputs, especially for high-stakes decisions.
  • Not a substitute for thinking. It executes your process at scale, but you still design the process.
  • Not magic. Garbage in, garbage out. The quality of your Promptlet depends on the quality of your prompt.
  • Not a data harvester. Prompt It does not collect analytics, track usage, or store your content remotely.
