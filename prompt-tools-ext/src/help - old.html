Overview

What Prompt It! does

  • Runs reusable AI promptlets on any selected text via the right-click context menu.
  • Shows responses in a dedicated side panel with copy, chaining, and token-usage details.
  • Ships with a curated set of defaults (Summarise, Rephrase, Verify, Max Reasoning, Learn a Language, Recipe Creator, Nutrition Analyser, Crossword Solver).
  • Lets you create, clone, reorder, export, and import your own promptlets.

Key flows

  • Select text ‚Üí right-click ‚Üí choose a promptlet ‚Üí view results in the side panel.
  • Open the toolbar popup for quick links to Manage Promptlets, Studio, and this Help center.
  • Use the side panel‚Äôs üîó Chain menu to run another promptlet on either the entire output or a highlighted snippet.
  • Back up or migrate custom promptlets with the Export and Import modals in the manager.

Fine-tuning

Advanced Settings

Every promptlet can override API parameters. Open Advanced Settings in the editor to adjust:

  • Model: GPT-5 Mini (default; fast, balanced, strong reasoning for most tasks), GPT-5.2 (latest; highest overall quality and reliability), GPT-5.1 (advanced; deep reasoning for complex analysis), GPT-5 Nano (ultra-fast; short, mechanical tasks only), GPT-4o (legacy; general-purpose writing).
  • Temperature: 0‚Äì2. Lower values produce more focused and deterministic output; higher values increase creativity and variation.
  • Max Tokens: Controls the maximum length of the response. Increase if output is truncated; higher values use more tokens and may increase cost.
Reminder: Model, temperature, and token settings are stored per promptlet, so you can mix fast utilities with deeper analysis tools.

Safety net

Export & Import

Export

  • Click Export on the manager page. Choose All Promptlets or Selected Promptlets (custom ones only).
  • Set a filename (downloads as .pi JSON) and confirm. Defaults are excluded because they can be reset anytime.
  • Selection mode includes Select All / Deselect All shortcuts so you can curate a shareable set quickly.

Import

  • Click Import, pick a .pi file (up to 5 MB), and review the preview list before confirming.
  • Name conflicts auto-resolve with (Copy) suffixes; built-in promptlets in the file are skipped to avoid duplicates.
  • If the file is empty or invalid, an error banner appears so you can choose another file.

Data handling

Privacy & Security

  • Local storage: API keys, promptlets, Studio content, and layout/theme preferences live in chrome.storage.local (or localStorage fallback).
  • Network usage: Selected text and prompts are sent only to OpenAI‚Äôs API endpoint for processing.
  • Scoped actions: Context menus appear only when text is selected, reducing accidental sends.
  • Reset safety: Resetting default promptlets does not touch your custom promptlets or API key.

Fix it fast

Troubleshooting

  • Missing API key or auth errors: Open Manage Promptlets and save a valid sk- key. Error cards in the side panel include a direct link.
  • Context menu is empty: Toggle any promptlet‚Äôs active switch off/on to rebuild the menu. Reinstall defaults if you removed them.
  • Output cuts off: Increase Max Tokens for that promptlet in Advanced Settings.
  • Slow responses: Switch to GPT-5 Mini or GPT-5 Nano for faster, cheaper runs; shorten the input.
  • Import fails: Ensure the file is under 5 MB and uses the .pi format exported from Prompt It!.
  • Service worker issues: Reload the extension from the extensions page to refresh background scripts.

Quick answers

FAQ

How do I create or edit promptlets?
Open Manage Promptlets, click Add New Promptlet for a fresh entry, or Clone a default to start with a template.
How do I chain promptlets?
After a promptlet finishes in the side panel or Studio, click üîó Chain and choose another promptlet. Highlight specific text first to chain only that part.
Can I restore the defaults?
Yes. Use üîÑ Reset Default Promptlets in the manager. Your custom promptlets and API key stay intact.
Which models can I use?
Each promptlet can target GPT-5.2, GPT-5.1, GPT-4o, GPT-5 Mini, or GPT-5 Nano. Adjust per promptlet inside Advanced Settings.
Does Studio keep my work?
Yes. Studio saves input, output, layout, and theme choices locally so the workspace reopens with your last state.
