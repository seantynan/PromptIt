Overview

Prompt It! is a browser extension that lets you run reusable AI ‚Äúpromptlets‚Äù on any text you select. Highlight text anywhere on the web, choose a promptlet (e.g., Summarise, Rephrase, Verify), and the result appears in a dedicated side panel. You can build your own promptlets, adjust model settings, and chain multiple promptlets together without leaving the page.

  • Works from the right-click context menu and the extension icon menu.
  • Stores your OpenAI API key locally only; it is used solely to call the OpenAI Responses API.
  • Ships with sensible defaults and lets you create, clone, and fine-tune your own promptlets.

Install & Set Up

  1. Install the extension.
  2. Open the management page. Click the Prompt It! toolbar icon and choose Manage Promptlets (or click the icon itself). This opens manage.html.
  3. Add your OpenAI API key.
    • Scroll to the OpenAI API Key section, paste your key (format sk-...), and click Save API Key.
    • Your key stays in local extension storage; it is never shared beyond calls to OpenAI.
  4. Confirm default promptlets. The built-in set (Summarise, Rephrase, Verify, Max Reasoning, Learn a Language, Recipe Creator, Nutrition Analyser, Crossword Solver) installs automatically on first run.
  5. Optional: Pin the extension icon so the popup menu (Manage, Studio, User Guide) is always one click away.

Quick Start

  1. Select text on any web page.
  2. Right-click and hover over Prompt It! in the context menu.
  3. Choose a promptlet (e.g., Summarise). The side panel opens and processes the selection.
  4. Read the output in the side panel. Use Copy Output or chain another promptlet directly.

Running Promptlets

Context Menu

  • Highlight text, right-click, choose Prompt It!, then pick a promptlet.
  • If no promptlets appear, ensure they are active in Manage Promptlets and rebuild the context menu by toggling their active status.

Extension Icon Popup

  • Click the toolbar icon ‚Üí Manage Promptlets opens the editor; Studio opens the themed scratchpad where the user can enter text and apply Promptlets; User Guide opens this page.
  • The popup is a launcher; promptlets still run from the context menu or side panel.

Pending Promptlets

  • If the side panel opens slowly, Prompt It! remembers the last request for a few seconds and runs it when the panel is ready.

Managing Promptlets

Use Manage Promptlets to enable, clone, create, or reset promptlets.

Default Promptlets

  • View-only set installed on first use. Click the clone icon to duplicate and edit.
  • Use Reset Default Promptlets to restore the factory set if you removed them.

Your Promptlets

  • Add New Promptlet: Opens the editor with emoji, name, prompt body, and advanced settings.
  • Edit: Update any field; the name is required. Prompts support up to 7000 characters.
  • Activate/Deactivate: Toggle to control whether a promptlet appears in the context menu and chaining menu.
  • Delete: Remove a custom promptlet permanently (defaults can be reset instead).

Advanced Promptlet Settings

Expand Advanced Settings in the editor to fine-tune API parameters for each promptlet.

  • Model: Choose GPT-5.1 (deep reasoning), GPT-4o (balanced), GPT-5 Mini (fast), or GPT-5 Nano (fastest/lightweight).
  • Temperature: 0‚Äì2. Lower is more deterministic; higher is more creative.
  • Max Tokens: 100‚Äì16000. Higher values allow longer responses but consume more quota.

  • Local storage: API keys and promptlets are stored in chrome.storage.local only.
  • Network usage: Selected text and prompts are sent only to OpenAI‚Äôs API endpoint for processing.
  • Context menu scope: Promptlets appear only when text is selected, reducing accidental sends.
  • Reset: Use Reset Default Promptlets to restore defaults without touching your customs or API key.

Troubleshooting

  • No output / Missing API key: Open Manage Promptlets and ensure your key is saved. Errors display in the side panel with a link to the manager.
  • Context menu missing promptlets: Toggle a promptlet‚Äôs active status in the manager to rebuild menus; reinstall defaults if needed.
  • Output is cut off: Increase Max Tokens in the promptlet‚Äôs advanced settings.
  • Slow responses: Use GPT-5 Mini or GPT-5 Nano for faster, cheaper runs; shorten input text.
  • Chaining runs on wrong text: Highlight the specific section in the side panel before clicking üîó Chain.
  • Permissions errors: Reload the extension from the extensions page to refresh service workers.

FAQ

Where do I edit or create promptlets?
Go to Manage Promptlets. Clone a default promptlet to start quickly or click Add New Promptlet.
How do I chain promptlets?
After a promptlet finishes, click üîó Chain in the side panel and pick the next promptlet. Select specific text first to chain only that snippet.
Can I restore the defaults?
Yes. In Manage Promptlets, click Reset Default Promptlets. Your custom promptlets remain untouched.
What models are available?
Promptlets can target GPT-5.1, GPT-4o, GPT-5 Mini, or GPT-5 Nano. Choose per promptlet in advanced settings.
Does the Studio save my text?
Yes. The Studio will save both input and output to local storage. This text is preserved after closing the Studio page.
